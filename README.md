# English to Ferb Latin Translator

A GUI App built using <strong>Vanilla JS, HTML, CSS</strong>. Hosted on codesandbox. 
This app takes an input as text in English and translate it to Ferb Latin language.
During their routine summer fun, the two brothers Phineas and Ferb(in a Dinsney series) 
create their own language, Ferb Latin which quickly catches on.
